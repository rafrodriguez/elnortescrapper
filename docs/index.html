Scrapper

This module implements the Scrapper class, which is a web scrapper for real estate classified ads of El Norte Avisos de Ocasión.

class scrapper.Scrapper(verbose=False)[source]¶

Web scrapper for real estate classified ads of El Norte Avisos de Ocasión www.avisosdeocasion.com website. It works with the categories of houses for sale in states of Mexico.

Example

>>> from reformascrapper import Scrapper
>>> my_scrapper = Scrapper()
>>> df = my_scrapper.scrap('venta_casas_cdmx', ad_limit=100)
